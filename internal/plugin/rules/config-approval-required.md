# Config Changes Require Approval ⚠️

Never modify without approval: eslint.config.*, .prettierrc*, tsconfig.json, vitest.config.*, next.config.*, etc.

**Forbidden:** Disabling type safety rules like `@typescript-eslint/no-explicit-any`

**Process:**
1. Report issue to user
2. Propose options with trade-offs
3. Wait for approval
4. Document in commit
